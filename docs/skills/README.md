# Skills Layout

This folder records the project skills that may be used by agents.

## Current Skill Categories

- `nextjs.md`: Next.js App Router work under this repository's version-specific docs rule.
- `supabase.md`: database migrations, auth, RLS, and seed workflows.
- `git.md`: branch, commit, sync, and push workflow.
- `security.md`: confidential storage, secret handling, and exposure prevention.
- `ocr.md`: Azure Document Intelligence + OpenAI normalization pipeline.
- `fx.md`: BNR/ECB exchange rate fetch, cache, and snapshot workflow.

## Skill Rule

Skills do not override `DEV_RULES.md`. They provide task-specific workflows only.
